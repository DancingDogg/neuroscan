About NeuroScan

NeuroScan is a Final Year Project developed at Universiti Tunku Abdul Rahman (UTAR) under the Bachelor of Information Technology (Honours) Communications and Networking programme. It leverages ensemble deep learning to detect ischemic stroke from brain MRI scans, combining multiple state-of-the-art architectures with explainable AI for transparent, trustworthy predictions.

PyTorch Flask Firebase ResNet50/101 DenseNet121/169 EfficientNet-B3 Vision Transformer Grad-CAM Attention Rollout Ensemble Learning

🎯 Objectives

  • Fine-tune and evaluate multiple state-of-the-art deep learning architectures
  • Develop an ensemble model combining CNN and transformer predictions
  • Implement Explainable AI (Grad-CAM, Attention Rollout) for transparency
  • Deploy the system as a secure, role-based web application

📊 Dataset

Source Hospital Pengajar UPM (HPUPM)
Training images 15,120 (balanced)
Validation images 38
Test images 34
